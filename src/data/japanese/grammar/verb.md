# 文法
1. 一类动词，动1，五段动词： 词尾在う段 / 3段上　→ 词尾う段变い段，3变2段，+　ます/ません

* 始まる　→　始まります 
* 終わる　→　終わります
* 働く　→　働きます  
* 行く　→　行きます

2. 二类动词，动2，一段动词：词尾为い段/え段 +　る （2段/4段+る）→　去る +　ます/ません

* 起きる　→　起きます
* 寝る　→　寝ます
* 食べる　→　食べます

3. 三类动词，动3 / 变格动词。 サ变　カ变

* する　→　します　しません
* 勉強する　→　勉強します
* 来る（くる）　→　来ます　（きます）
